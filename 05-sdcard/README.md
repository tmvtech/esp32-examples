This project sets up a SD Card via SPI to read and write files to.

## Pins

| Card Reader       | Esp         |
| ----------------- | :---------- |
| MOSI (DI)         | D23         |
| MISO (DO)         | D19         |
| SCK (CLK)         | D18         |
| CS (SS)           | D2          |
| VCC               | 3.3v        |
| GND               | GND         |
