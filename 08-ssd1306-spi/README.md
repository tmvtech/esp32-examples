This project runs an oled display controlled by a SSD1306 chip which communicates with the Esp32 via SPI.

## Pins

| Display           | Esp         |
| ----------------- | :---------- |
| MOSI (D1)         | D23         |
| SCK (D0)          | D18         |
| CS (SS)           | D2          |
| DC (SS)           | D4          |
| RST               | D5          |
| VCC               | 3.3v        |
| GND               | GND         |
